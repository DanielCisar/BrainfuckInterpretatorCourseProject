Проектът реализира интерпретатор за езика Brainfuck. Идеята е да се превърне текстов или бинарен Brainfuck код в изпълнима програма, която оперира върху безкрайна лента. Допълнително са имплементирани оператори над програмите (конкатенация, паралелно изпълнение, редуване) и инструмент за синтез на кратки програми от входно-изходни примери.
